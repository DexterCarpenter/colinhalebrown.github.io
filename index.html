Universal Arduino Controller

This was my first big design project using my 3d printer to make an arduino controller. I wanted to design a controller from start to finish so I could learn how to program a transmitter and a reciever along with button and joystick inputs. I started designing the controller in the computer aided-design software fusion 360. My goal for the hardware was to fit a battery along with the arduino and input shield. I didn't want the controller to be too big or bulky so I assembled the parts and moved them around until I found a configuration I liked. I settled on a configuration that puts the battery under the overhang of the large input shield. I made three iterations of the controller before finalizing the design. and built it using an arduino and input shield.
